数组遍历方法

发表于 2021-10-20 分类于 前端 阅读次数： Valine：

reduce 求数组和

  1. 语法：

     arr.reduce(function(prev,cur,index,arr){
 ...
 }, init);

  2. 回调里面有4个参数：

    • prev 表示上一次调用回调时的返回值，或者初始值 init;
    • cur 表示当前正在处理的数组元素；
    • index 表示当前正在处理的数组元素的索引，若提供 init 值，则索引为0，否则索引为1；
    • arr 表示原数组；
    • init 表示初始值。

  3. 传入初始值

  4. 不传初始值

jquery 遍历数组

  1. 只能运用于jquery语法中，i是数组的索引，x是数组的值.

forEach 遍历数组

  1. 无return 即使有return，也不会返回任何值，并且会影响原来的数组
  2. 有三个默认值：
  • 第一个 当前数组值
  • 第二个 索引
  • 第三个 就是原数组

filter 过滤数组

  1. 过滤数组，返回一个满足要求的数组

  2. 有两个默认值

    • 第一个当前索引值
    • 第二个索引

    map 遍历数组

  3. 映射数组(遍历数组),有return 返回一个新数组

  4. 有三个默认值：

    • 第一个 当前数组值
    • 第二个 索引
    • 第三个 就是原数组

every 判断数组全满足条件

  1. 依据判断条件，数组的元素是否全满足，若满足则返回ture

  2. 有两个默认值：

    • 第一个当前索引值
    • 第二个索引

      some 遍历判断数组满足条件

  3. 依据判断条件，数组的元素是否有一个满足，若有一个满足则返回ture

  4. 有两个默认值：

    • 第一个当前索引值
    • 第二个索引

Ps : Array.from() 将伪数组变成数组，就是只要有length的就可以转成数组
